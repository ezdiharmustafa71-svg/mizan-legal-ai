# ⚖️ Mizan Legal AI

Mizan Legal AI is a **Contract Intelligence & Decision Engine** designed to analyze legal agreements and support structured decision-making.

## 🎯 Vision
To build an AI-powered legal assistant that evaluates contracts like a professional lawyer by identifying risks, obligations, and decision points.

## 🧠 Core Features (MVP)
- Contract text extraction
- Clause detection (e.g., liability, exit, profit sharing)
- Risk scoring system
- Decision output (Sign / Do Not Sign / Review)
- Recommendation engine

## 🏗️ Architecture
- Python (Core logic)
- Streamlit (UI)
- OpenAI API (AI reasoning)
- Rule-based decision engine

## 🚀 Goal
To assist legal professionals and businesses in faster, more consistent contract evaluation.

## 📌 Status
Early MVP stage (in development)

⚖️ Built with focus on legal intelligence and decision systems, not just text generation.
